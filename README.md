# 3B-Balistik-Simülasyon
RK4 (Runge-Kutta 4) algoritması kullanılarak geliştirilmiş, 3 boyutlu balistik simülasyon terminali. Hava yoğunluğu (ISA), Coriolis, Eötvös, Magnus ve Spin Drift etkilerini dinamik olarak hesaplar. .50 BMG ve .338 Lapua gibi mühimmatlar için yüksek hassasiyetli yörünge analizi sunar (eğtim amaçlıdır).
